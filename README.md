# Discord Music Bot

## Environment

- Python 3.10

### Environment setup steps

1. Install `virtualenv` to keep develop environment isolated
   
   ```
   pip install virtualenv
   ```

2. Create an virtual environment
   
   ```
   virtualenv venv
   ```

3. Active the virtual environment
   
   ```
   source venv/bin/activate
   ```

4. Install all require libraries
   
   ```
   pip install -r requirements.txt	
   ```

5. Create an `.env` file so music player could search from youtube's api through your token
